# Параметры и аргументы

## Новая задача

Заказчик доволен тем, как выводятся рекламные сообщения. Но теперь он хочет, чтобы **текст** рекламы менялся (несколько разных рекламных сообщений).

### Неправильный подход

Можно было бы создать три разные функции: `show_ads_1`, `show_ads_2`, `show_ads_3`.
Но тогда мы снова нарушим принцип **DRY**. Если мы захотим поменять волнистую линию `~~~` на звездочки `***`, нам придется править это в трех разных функциях.

### Правильный подход: Передача данных

Нам нужно оставить **одну** функцию, но научить её принимать сообщения извне. Представим, что функция — это цех на фабрике. Процесс в цехе происходит один и тот же, но "сырье" (входные данные) используются разные. Результат работы цеха (вывод рекламы) будет немного отличаться. 

Для этого используются **параметры**.

### Шаг 1. Создаем "место" для данных (параметр)

При создании функции мы можем указать **переменную** в скобках. Это как контейнер для сырья в цехе

```python
# text — это ПАРАМЕТР (переменная, используется в теле функции)
def show_ads(text):
    print()
    print("~~~")
    print(text)  # выводим то, что передадут при вызове 
    print("~~~")
    print()

```

Теперь функция `show_ads` не знает заранее, что она выводит. При вызове этой функции программист обязан передать в скобках объект, он будет присвоен переменной `text`. 

### Шаг 2. Передаем данные (аргумент)

Объект, который мы напишем в скобках при вызове- это **аргумент** (значение параметра). Если в объявлении функции есть параметр (переменная в скобках), то при вызове функции мы обязаны передать соответствующий аргумент (значение).

```python
# "Скидки в школе магии!" — это АРГУМЕНТ (конкретное значение)
show_ads("Скидки в школе магии!")
```

### Итоговый код игры

Посмотрим, как работает одна и та же функция с разными данными:

```python
def show_ads(text):
    print()
    print("~~~")
    print(text)
    print("~~~")
    print()

print("Добро пожаловать в игру!")

# Вызов 1: передаем в функцию рекламный текст
show_ads("Покупайте наши курсы!")

print("Вы перешли на новый уровень!")

# Вызов 2: передаем в ту же функцию другой текст
show_ads("Мечи и щиты за полцены!")

print("Игра окончена!")

# Вызов 3: опять новый аргумент
show_ads("Яндекс - найдется всё!")
```
