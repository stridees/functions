# Функции
**Вопросы:** 

* В чем суть принципа DRY? Какие проблемы возникают при нарушении этого принципа?
* Что такое функция? В чем плюсы использования функций?

<details>
<summary>Ответы</summary>

**DRY** (**D**on't **R**epeat **Y**ourself) — принцип "не повторяйся". Необходимо избегать копирования своего кода. Если есть повторяющиеся строки, при необходимости внести изменения придется делать это во множестве мест.  

**Функция** (подпрограмма) — именованный блок кода, который можно написать один раз и использовать многократно. Вместо нескольких строк в коде пишется короткое имя функции. Это упрощает процесс внесения изменений, повышает читаемость, код занимает меньше места.
</details>

**Навыки:**
* Объявление функции (имя, тело)
* Вызов функции

Ниже фрагмент программы-игры. В разных местах игра показывает рекламу, и каждый раз это сделано копированием одного и того же блока кода.

```python
print("Добро пожаловать в игру!")

# показ рекламы
print()
print("~~~")
print("Яндекс - найдется всё!")
print("~~~")
print()

# <здесь много строк кода - в игре что-то происходит>
print("Вы перешли на новый уровень!")

# показ рекламы
print()
print("~~~")
print("Яндекс - найдется всё!")
print("~~~")
print()

# <еще код игры>
print("Игра окончена!")
print("Вы проиграли!")

# показ рекламы
print()
print("~~~")
print("Яндекс - найдется всё!")
print("~~~")
print()
```

Когда понадобится изменить оформление/текст рекламы, придется делать правки множестве мест, это неудобно. Нужно будет вспомнить, где находятся скопированные фрагменты, сколько их, внимательно внести изменения в каждый, не упустив ни одного блока. В реальной жизни редактировать программу приходится очень часто, поэтому DRY является важным принципом.

### Решение с функциями

Определим функцию `show_ads`, которая содержит код вывода рекламы, и будем вызывать её в нужных местах.

```python
# определение функции (def - define)
def show_ads():
    print()
    print("~~~")
    print("Яндекс - найдется всё!")
    print("~~~")
    print()

print("Добро пожаловать в игру!")

# показ рекламы
show_ads()  # вызов функции

# <здесь много строк кода - в игре что-то происходит>
print("Вы перешли на новый уровень!")

# показ рекламы
show_ads()  # вызов функции

# <еще код игры>
print("Игра окончена!")
print("Вы проиграли!")

# показ рекламы
show_ads()  # вызов функции
```

1. Найди повторяющийся код (блок музыкальной паузы).

3. Замени повторяющиеся строки на вызов твоей новой функции.

<summary>ПОСЛЕ выполнения проверь себя</summary>

Вот как выглядит программа с использованием функции. Код стал короче, а если мы захотим поменять "Ля" на "Бум", нам придется исправить это только в одном месте!

```python
# объявляем функцию
def play_music():
    for i in range(3):
        print("Ля")


# --- Основная программа ---

print("В эфире радио 'Python FM'!")
print("Прослушайте прогноз погоды...")

play_music()  # вызов функции

print("А теперь переходим к новостям.")
print("Опубликована новая версия...")

play_music()  # вызов функции

print("Продолжаем наш концерт по заявкам!")
```
