# Функции

Ниже приведен код фрагмента программы-игры. Периодически в игре должна отображаться реклама. 

```python
print("Добро пожаловать в игру!")

# показ рекламы
print()
print("~~~")
print("Яндекс - найдется всё!")
print("~~~")
print()

# <здесь много строк кода - в игре что-то происходит>
print("Вы перешли на новый уровень!")

# показ рекламы
print()
print("~~~")
print("Яндекс - найдется всё!")
print("~~~")
print()

# <еще код игры>
print("Игра окончена!")
print("Вы проиграли!")

# показ рекламы
print()
print("~~~")
print("Яндекс - найдется всё!")
print("~~~")
print()
```
В этой версии программист просто скопировал один и тот же блок кода несколько раз. Программа работает, но очень скоро могут возникнуть трудности.

Когда понадобится изменить оформление/текст рекламы, придется делать правки во множестве мест. Нужно будет вспомнить, где находятся скопированные фрагменты, сколько их, внимательно внести изменения в каждый, не упустив ни одного блока. В реальной жизни редактировать программу приходится очень часто, она занимает сотни и тысячи строк, размещается в разных файлах. Подобные действия превращаются в мучительную рутинную работу, где велик риск заменить код не во всех скопированных фрагментах из-за ошибки, невнимательности.

### Решение с функциями

Определим (создадим, опишем) функцию `show_ads`, которая содержит код вывода рекламы, и будем вызывать её в нужных местах.

```python
# определение функции (def - define)
def show_ads():
    print()
    print("~~~")
    print("Яндекс - найдется всё!")
    print("~~~")
    print()

print("Добро пожаловать в игру!")

# показ рекламы
show_ads()  # вызов функции

# <здесь много строк кода - в игре что-то происходит>
print("Вы перешли на новый уровень!")

# показ рекламы
show_ads()  # вызов функции

# <еще код игры>
print("Игра окончена!")
print("Вы проиграли!")

# показ рекламы
show_ads()  # вызов функции
```
Сравните это решение с предыдущим. Очевидно, что код стал занимать меньше места. Писать короткое `show_ads()` проще, чем каждый раз размещать несколько строк кода. Мы описали показ рекламы один раз. Главное, чего мы добились - изменять рекламное объявление и способ его показа нужно **в одном месте**, в блоке `def`. Изменения будут учтены для всех показов, даже если их будет очень много.

Функция - это **подпрограмма** ("программа в программе"), которую сначала объявляют (описывают), а затем вызывают (запускают содержащиеся в ней инструкции), в том числе многократно.

**Объявление функции** (блок `def`) содержит её имя и тело (то, что будет выполняться). Для имен функций обычно используют глаголы (`check_answer`, `count_price` и т.д.)
